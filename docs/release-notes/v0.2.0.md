# Artifactor v0.2.0

## Highlights

- **Configuration system**: YAML-based config with discovery (`artifactor.yml`)
- **Config CLI**: `artifactor config validate` and `artifactor config print` commands
- **Deterministic adapter registry**: Explicit adapter registration and debugging tools
- **Forced adapter semantics**: `--adapter` flag with fallback behavior for unsupported sources
- **Repo-root packaging**: Modern `pyproject.toml` setup enabling `pip install artifactor`
- **Trusted Publishing**: GitHub Actions workflow for secure PyPI releases via OIDC

## Installation

```bash
pip install artifactor
```

## Quick Start

```bash
# Ingest URLs from a file
artifactor ingest --urls urls.txt

# Validate configuration
artifactor config validate

# List available adapters
artifactor adapters list
```
